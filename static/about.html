About RecipeVault

The Project

RecipeVault is a full-stack web application designed as a portfolio project to showcase modern backend development skills. The API is built with Python and FastAPI, featuring JWT authentication, full CRUD functionality for user-owned recipes, and a PostgreSQL database managed by Docker and Alembic for migrations.

The Developer

This project was developed by Priel Krishtal, a computer science student passionate about building robust and scalable backend systems.
